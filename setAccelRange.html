setAccelRange()

Description
Set the range for the acceleromater to one of the preset values.
ValueRange

LIS3DH_RANGE_8_G+/- 8g
LIS3DH_RANGE_4_G+/- 4g
LIS3DH_RANGE_2_G+/- 2g
Syntax
setAccelRange(range);
Parameters
range [lis3dh_range_t]: The range to use (see table above).
Returns
Nothing
